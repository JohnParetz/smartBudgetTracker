# Smart Budget Tracker

## Overview
Smart Budget Tracker is a web application that allows users to track income and expenses, view their financial status, and receive personalized budget feedback.

---

## What the App Does

- Users can add income and expense entries  
- The app calculates and displays total income, total expenses, and remaining balance in real time  
- All transactions are listed and can be deleted individually  
- Users can clear all transactions at once  
- Data is saved using localStorage, so it remains after refreshing the page  

---

## Pages

### Home
- Add income and expenses  
- View totals and transaction list  

### Health Check
- Analyzes financial data  
- Displays whether the user is in good or poor financial health  
- Shows animated feedback and advice  

### Profile
- Users can create multiple profiles  
- Set savings goals and balances  
- Compare progress toward goals  
- Delete saved profiles  

### Suggestions
- Provides budget tips based on user data  
- Displays different suggestions with colors and animations  

### About
- Explains the purpose of the app and tools used
